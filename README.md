# 请假系统
分为两个项目，Web主项目，微信子项目  
独立开发给企业私人定制的一个请假系统，亮点: 流程、Excel批量导入导出、邮件/微信定时通知及快捷审批.  
前端架构：jQuery、Bootstrap  
后端架构：jfinal、POI、Java Mail、dom4j、fastjson、Mysql
